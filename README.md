# CLI H3C Client

A CLI H3C Client in \*NIX

## Usage

Just run make under the root directory of project.

```bash
make
```

Then you get `clih3c` executable file.

```bash
-h --help        print help screen
-u --user        user account
-p --password    password
-i --iface       network interface (default is eth0)
-d --daemonize   daemonize
```
